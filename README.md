# 氷焔雀譚 - Character Mahjong PWA

うちの子たちと遊ぶ、キャラゲー寄り四人麻雀PWAプロトタイプです。v13では立ち絵画像のファイル名を固定し、assets/characters に置くだけでキャラ画像として表示されるようにしました。

## 入っているもの

- `index.html`：画面本体
- `style.css`：雀魂っぽい雰囲気のUI
- `app.js`：牌山、配牌、ツモ、打牌、ツモ/ロン/立直、AI処理
- `manifest.json`：PWA設定
- `sw.js`：オフラインキャッシュ用Service Worker
- `assets/icon.svg`：PWAアイコン
- `assets/characters/`：キャラ立ち絵PNG置き場
- `CHARACTER_IMAGE_FILENAMES.txt`：立ち絵の固定ファイル名一覧

## 実装済み

- 四人打ちプロトタイプ
- 操作キャラ選択
- 対戦相手3人選択
- 朱音・梁泫・煜・瑤・マクシミリアンなど強キャラAI
- ツモ / ロン判定
- 七対子、国士無双、通常和了形の判定
- 立直
- ドラ簡易計算
- カットイン、ログ、キャラカード
- 初心者向け捨て牌ガイド、おすすめ役表示、危険度メモ
- PWA形式（HTTPSやlocalhostで開くとインストール可能）

## まだ簡易実装

- 点数計算は雰囲気寄せの簡易式
- 東1局のみ
- 役判定は最小限
- 本物の雀魂素材は未使用（著作権的に安全なオリジナルUI）

## 起動方法

直接 `index.html` を開いても遊べますが、PWAとして入れるならローカルサーバーで開いてください。

```bash
cd minaho_mahjong_pwa
python -m http.server 8000
```

そのあとブラウザで以下を開きます。

```text
http://localhost:8000
```

Chromeならアドレスバー付近から「インストール」できます。
